# Workshop JSON assets (Cursor / maintainers)

These folders back the **embedded** JSON in [`workshop/prompts/`](../../../workshop/prompts/) and the Cursor agent skill at [`../SKILL.md`](../SKILL.md).

**Attendees** using Port in the browser should follow [`workshop/README.md`](../../../workshop/README.md)—they do not need to open this directory.

After editing any file here, run from the repo root:

```bash
python3 workshop/scripts/sync-prompt-embeds.py
```
