# Step 0 — Portal Branding

## What We're Doing

Making the portal feel like PlatformCon-Carne's. Small step, big signal.

## What to Build

Update the portal name to `PlatformCon-Carne Platform` via Port settings or MCP.

## Why It Matters

When developers land on the portal for the first time, they need to know this is *their* platform team's product — not a generic tool. Ownership starts with branding.
